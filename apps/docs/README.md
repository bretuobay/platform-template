# Docs App Starter

Next.js docs app that documents architecture boundaries and implementation workflow for this template.

## Focus Areas

- Package-first feature development
- API contract sharing between web and edge worker
- Tailwind v4 token and utility usage from `@repo/brand`

## Commands

```bash
npm run dev --workspace docs
npm run build --workspace docs
npm run lint --workspace docs
npm run typecheck --workspace docs
npm run test --workspace docs
```
