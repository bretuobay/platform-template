# Web App Starter

Next.js starter app for product-facing UI.

## Includes

- Marketing surface (`app/(marketing)`)
- Authenticated shell (`app/(app)`)
- Starter dashboard route composed from `@repo/feature-dashboard`
- Shared brand tokens and classes from `@repo/brand/styles.css`

## Commands

```bash
npm run dev --workspace web
npm run build --workspace web
npm run lint --workspace web
npm run typecheck --workspace web
npm run test --workspace web
```
